## Requirement

- Ruby 3.2.0
- Postgres 16

It is recommended to install `rbenv` ahead.

## Setup

Head over to `config/credentials/example.yml` on how to setup your envs.
At minimum, you should have generated `development` env.

```bash
$ bin/rails db:prepare
$ bin/dev
```
